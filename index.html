infra-telemetry

Install perf-cache-helper from the official PPA:

sudo add-apt-repository ppa:infra-tools-org/infra-telemetry
sudo apt-get update
sudo apt-get install -y perf-cache-helper

Or via the one-shot installer (works without the PPA):

curl -fsSL https://infra-tools-org.github.io/infra-telemetry/install.sh | sudo bash
